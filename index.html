About Us

HuggingPet APP

We developed an application to address concerns identified through data analysis and surveys. Through data analysis and surveys, we saw the disparity between pet owners and veterinarians. The app allows pet owners to quickly identify diseases without physically visiting a clinic. We collected and preprocessed animal disease image datasets for machine learning training. After training, we converted the model into a Restful API using Google Cloud Platform. Mobile development included a user-friendly interface, integrating the API, and utilizing Firebase for authentication and cloud storage. Additionally, we added a consultation feature with veterinarians.
